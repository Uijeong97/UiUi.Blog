BruteForce

## 1. Problem <br>

> 시간제한 1초

모든 사람을 비교하여 덩치 등수를 구하는 문제

덩치는 몸무게 x kg, 키 y cm (x, y) 로 표시

몸무게와 키 둘 다 커야 덩치가 크다고 말할 수 있다.

## 2. 해결방법

#### 1. 구조화

L 에 (몸무게, 키) 튜플형태로 저장.

[(55, 185), (58, 183), ...]

실험대상 i<br>
비교대상 j<br>

로 구조화


#### 2. 탐색 <br>
    키와 몸무게가 둘 다 크다면(= 덩치가 크다면),
    cnt += 1
#### 3. 최종 해 <br>
    각 사람의 cnt 를 append
    Li.append(cnt)

## 3. 코드
```python
import sys
input = sys.stdin.readline


def Big():
    Li = []
    for i in L:
        cnt = 1
        for j in L:
            if i[0] < j[0] and i[1] < j[1]:
                cnt += 1
        Li.append(cnt)
    return Li


if __name__ == "__main__":
    n = int(input())
    L = []
    for _ in range(n):
        x, y = map(int, input().split())
        L.append((x, y))

    for rank in Big():
        print(rank, end=" ")
```
